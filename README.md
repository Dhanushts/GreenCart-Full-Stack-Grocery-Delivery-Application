# GreenCart-Full-Stack-Grocery-Delivery-Application
GreenCart is a full-stack grocery delivery web application built using React that allows users to browse, search, and purchase grocery products online, while providing an admin dashboard for managing products and orders. The application focuses on a smooth user experience, clean UI, and efficient product management.
